# 🚀 NewTab
**NewTab** est une extension de navigateur ultra-légère qui vous permet de reprendre le contrôle de votre page "Nouvel Onglet". Au lieu de la page par défaut souvent encombrée, elle vous redirige instantanément vers l'URL de votre choix.
## ✨ Fonctionnalités
 * ⚡ **Redirection Instantanée** : Ouvrez Google, votre tableau de bord personnel, ou n'importe quel site web dès que vous créez un nouvel onglet.
 * 🛠️ **Configuration Simple** : Définissez l'URL de destination dans les options de l'extension.
 * 🎈 **Poids Plume** : L'extension est conçue pour être la plus légère possible, sans aucun impact sur les performances de votre navigateur.
 * 🔒 **Respect de la vie privée** : Aucune donnée de navigation n'est collectée. L'extension ne fait qu'une seule chose : vous rediriger là où vous le souhaitez.
## ⚙️ Comment ça marche ?
L'extension utilise l'API chrome.tabs pour détecter l'ouverture d'un nouvel onglet et remplace l'URL chrome://newtab par celle que vous avez définie dans les paramètres. C'est la solution idéale pour :
 * Utiliser un moteur de recherche spécifique.
 * Atterrir directement sur votre calendrier ou votre liste de tâches.
 * Utiliser une page HTML locale personnalisée.
